# Space_CoBot
Code for the Space CoBot free flyer robot
Experimental Evaluation of the Space CoBot free-flyer robot on a ground testing setup


#Documentation
Code and hardware integration is documented in the projects drive. Especialy in the reference manual

### Usage
Consult the Robot's Reference Manual at the projects Drive
All the steps to operate this code are present in the Reference Manual

### Automation script
To automate some regular commands in the cobot operation the cdSpaceCobot_ws.sh file contains some shell scripts. It is commented accordingly and further documented in the reference manual.

### ROS code
The src files can be found in the following path:
/SpaceCobot-ros_px4/px4-ROS/ros_controllers/space_cobot_ws/src

The modules are documented in the reference manual
