Initialize:
- Pose = {Position, Orientation, Velocity}
- IMU_Biases = {AccelerometerBias, GyroscopeBias}

Main Loop:
While True:
    - Read IMU data {LinearAcceleration, AngularVelocity}

    - Correct IMU data for biases:
        LinearAcceleration_corrected = LinearAcceleration - AccelerometerBias
        AngularVelocity_corrected = AngularVelocity - GyroscopeBias

    - Integrate AngularVelocity to update Orientation:
        Orientation = IntegrateOrientation(Orientation, AngularVelocity_corrected, deltaTime)

    - Remove gravity from LinearAcceleration:
        LinearAcceleration_corrected = RemoveGravity(LinearAcceleration_corrected, Orientation)

    - Integrate LinearAcceleration to update Velocity and Position:
        Velocity = Velocity + LinearAcceleration_corrected * deltaTime
        Position = Position + Velocity * deltaTime

    - Update Pose:
        Pose = {Position, Orientation, Velocity}

    - Optional: Correct Pose using external references to reduce drift

    - Wait for next IMU reading
